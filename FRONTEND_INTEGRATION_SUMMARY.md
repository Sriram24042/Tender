## 🚀 **How to Run**

### **Quick Start**
```bash
# Install all dependencies
npm run install:all

# Start both frontend and backend
npm run dev
```

### **Access Points**
- **Frontend**: http://localhost:5173
- **Backend API**: http://localhost:8000
- **API Docs**: http://localhost:8000/docs
